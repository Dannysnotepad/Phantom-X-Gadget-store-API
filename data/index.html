Phantom-X Gadget

Welcome to Phantom-X Gadget: A Gadget store API for developers

What is Phantom-X Gadget API

Phantom-X is an easy-to-use, versatile Gadget Store CRUD REST API packed with dummy data for developers to use in the simplest way possible, especially those learning the art of consuming APIs.

How Does it Work?

A simple GET request can be used to fetch data from our API via a browser or CURL.

Creating / Adding Products:


You can add products by sending a POST request with the instructions below

When adding a product, the header content-type must be JSON and the JSON sent through the body parameters must follow this order id, img_url, item, category, description and price

For example

        
  {
  "id": "11",
  "img_url": " ",
  "item": "Honps Digital Watch",
  "category": [
  "lightweight",
  "pokectfriendly",
  "tech",
  "digital watch"
  ],
  "description": "Digital wrist watch with sleek design and advance tech components",
  "price": 50.9
  }

Reading / Displaying Products:


Request Product by Category

URL: https://phantom-X-gadget.vercel.app/api/products/category/{category}

Example: https://phantom-X-gadget.vercel.app/api/products/category/pokectfriendly

Returns a products having the same or in that category requested, most of the time you get more than one products because they share that to category.


Request Product by ID

URL: https://phantom-X-gadget.vercel.app/api/products/{id}

Example: https://phantom-X-gadget.vercel.app/api/products/8

Returns a product with ID requested, current number of products in the database is 10.


Request Every Products

URL: https://phantom-X-gadget.vercel.app/api/products

This will compact all available data on in the Data base and send them as a JSON Object.

Updating Products


You can update products info by sending a PUT request to the url below and also specify the product you want to update through the id, and also the field you want to update.

URL: https://phantom-X-gadget.vercel.app/api/products/{id}

              
        {
          "price": 105
        }

Deleting / Removing Products


You can delete products by sending a DELETE request followed buy the id of the product you want to delete

URL: https://phantom-X-gadget.vercel.app/api/products/{id}
